LEDBox
======

Arduino YUN LEDBox


Libs Needed:
https://github.com/gauravmm/HT1632-for-Arduino


Connection
Pin 4 - HT1632c CLK
Pin 5 - HT1632c CS
Pin 7 - HT1632c DATA
Pin 6 - HT1632c WR

Pin 8 - Button Red
Pin 10 - Button Yellow
Pin 11 - Button White
Pin 12 - Button Green
Pin 3 - Button Blue
Pin 2 - Button Black

Pin 9 - Speaker
